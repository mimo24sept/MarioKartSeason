MKWii Retro Rewind

MKWii Retro Rewind est un mod inédit qui modernise Mario Kart Wii. Redécouvrez des circuits rétro emblématiques, des visuels améliorés et des fonctionnalités supplémentaires. Ce mod est conçu pour fonctionner avec Dolphin Emulator et offre une expérience revisitée et optimisée.

Installation rapide :
1. Téléchargez la ROM et le launcher avec les liens ci-dessous.
2. Extrayez les fichiers dans un dossier dédié.
3. Lancez Dolphin Emulator et chargez la ROM.
4. Démarrez WeelWizard pour accéder aux circuits rétro.

Tutoriel d'installation

1. Téléchargement des fichiers nécessaires :
- Téléchargez la ROM de Mario Kart Wii (fichier `.rvz`).
- Téléchargez le launcher WheelWizard.
- Téléchargez Dolphin Emulator (dernière version disponible).

2. Configuration de Dolphin :
- Lancez Dolphin Emulator.
- Allez dans `Options` → `Configuration` → `Dossiers`.
- Cliquez sur `Ajouter` et sélectionnez le dossier où se trouve la ROM.
- Rendez-vous dans `Paramètres des manettes` et configurez les commandes.

3. Lancement initial de Mario Kart :
- Lancez Mario Kart Wii dans Dolphin pour tester la manette.
- Vérifiez que les commandes répondent correctement.
- Si tout fonctionne, créez un permis de jeu pour enregistrer votre progression.

4. Paramétrage de WheelWizard :
- Lancez le logiciel WheelWizard.
- Ouvrez les paramètres et renseignez les chemins indiqués en vert :
  • Chemin du jeu (ROM Mario Kart Wii).
  • Chemin du logiciel Dolphin.

5. Lancement du mod :
- Une fois les chemins configurés, retournez au menu principal de WheelWizard.
- Lancez le jeu via le bouton prévu à cet effet.
- Enjoy et bon drift !

Jouer en Multijoueur

1. Préparation des amis :
- Lancez le jeu avec WheelWizard et créez un permis si ce n'est pas déjà fait.
- Accédez au menu 'Friends' et notez votre code unique.
- Échangez vos codes d'ami avec les joueurs avec qui vous souhaitez jouer.
- Entrez leurs codes dans le même menu et attendez qu'ils fassent de même.

2. Connexion en ligne :
- Lancez WheelWizard et allez dans les paramètres de connexion.
- Vérifiez que votre connexion Internet est active et stable.
- Choisissez l'option `Connexion multijoueur` et cliquez sur `Se connecter`.
- Attendez que vos amis se connectent et rejoignez la même salle.

3. Lancement de la partie :
- Une fois connecté, choisissez vos personnages et circuits.
- Lancez la course et profitez des fonctionnalités multijoueurs en ligne !

4. Mode local et splitscreen :
- Le mode local en écran partagé reste disponible pour jouer jusqu'à 4 joueurs.
- Branchez les manettes supplémentaires et configurez-les dans Dolphin (`Paramètres des manettes`).
- Lancez une partie locale et amusez-vous à plusieurs sur le même écran.
